Element colors can be customized by specifying Primary and Accent colors.

Include FlThemeService in the app provider as a Component dependency. Use the set() method to establish the theme colors.

import { FlThemeService } from './wui/theme2/theme.service';

@Component({
  selector: 'app-root',
  ...
  providers: [FlThemeService]
})
export class AppComponent {

  constructor(private _theme: FlThemeService) {
    this._theme.set('teal-lightBlue-light');
  }
}

Dark text on light backgrounds

The level of opacity used for text depends on whether your background is dark or light. For dark text on light backgrounds, apply the following opacity levels:

Dark text Color / Opacity Hex color
Primary text #000000 / 87% #202020
Secondary text #000000 / 54% #737373
Disabled / Hint text #000000 / 38% #9B9B9B
Dividers #000000 / 12% #9B9B9B
Primary color #FFFFFF / 100%
Accent color #FFFFFF / 100%

White text on dark backgrounds

White text appearing on colored backgrounds should do so at an opacity of 100%.

Primary text #202020

Themes let you apply a consistent tone to an app. The theme specifies the darkness of the surfaces, level of shadow, and appropriate opacity of ink elements. Themes are dynamic based on chosen Primary and Accent colors as well as then further segmented by "light" or "dark."
