Drawer Title

Referred to as "side nav" in Google Material, the drawer is a common structural design element that allows content to be placed off screen and toggled into view.

Material design uses the drawer/side nav as primarily as navigation lists. The Flux spec extends the role to include further meta-data, relevent to the content of the screen.


An undocked Drawer. The Drawer can be canceled by clicking the overlay or pressing the Esc key.

Set align="end" to open the Drawer on the opposite side.
Name Type Default Description
align string start Aligns the drawer with the start of the Drawer Container (the left side).
disableClose boolean false Disables closing the Drawer with the Esc key.
mode string over Sets the behavior of the Drawer. over causes the Drawer to appear above the primary content. push causes the primary content to be pushed off screen. side will reduce the space of the primary content to allow the Drawer to float beside it.
opened boolean false Whether the Drawer is open.
open function Event emitted when the Drawer is fully opened.
open-start function Event emitted when the Drawer is being opened.
width string Give the Drawer a custom width when expanded.
